Description

GstQueueArray is an object that provides standard queue functionality based on an array instead of linked lists. This reduces the overhead caused by memory managment by a large factor.

gst_queue_array_find ()

guint               gst_queue_array_find                (GstQueueArray *array,
                                                         GCompareFunc func,
                                                         gpointer data);

Finds an element in the queue array, either by comparing every element with func or by looking up data if no compare function func is provided, and returning the index of the found element.

Note that the index is not 0-based, but an internal index number with a random offset. The index can be used in connection with gst_queue_array_drop_element(). FIXME: return index 0-based and make _drop_element() take a 0-based index.
