GStreamer Base Plugins 1.0 Library Reference Manual

FFT Library

gstfft — General FFT functions and declarations
gstffts16 — FFT functions for signed 16 bit integer samples
gstffts32 — FFT functions for signed 32 bit integer samples
gstfftf32 — FFT functions for 32 bit float samples
gstfftf64 — FFT functions for 64 bit float samples

The gstfft library is based on kissfft by Mark Borgerding.

This library should be linked to by getting cflags and libs from gstreamer-plugins-base-1.0.pc and adding -lgstfft-1.0 to the library flags.
