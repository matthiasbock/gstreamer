Description

VertigoTV is a loopback alpha blending effector with rotating and scaling.

Example launch line

1
 
gst-launch-1.0 -v videotestsrc ! vertigotv ! videoconvert ! autovideosink
This pipeline shows the effect of vertigotv on a test stream.
