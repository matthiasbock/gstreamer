rtpamrpay

rtpamrpay — Payload-encode AMR or AMR-WB audio into RTP packets (RFC 3267)

Synopsis

struct              GstRtpAMRPay;

Object Hierarchy

  GObject
   +----GInitiallyUnowned
         +----GstObject
               +----GstElement
                     +----GstRTPBasePayload
                           +----GstRtpAMRPay

Description

Payload AMR audio into RTP packets according to RFC 3267. For detailed information see: http://www.rfc-editor.org/rfc/rfc3267.txt

Example pipeline

1
 
gst-launch -v audiotestsrc ! amrnbenc ! rtpamrpay ! udpsink
This example pipeline will encode and payload an AMR stream. Refer to the rtpamrdepay example to depayload and decode the RTP stream.
